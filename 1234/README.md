## 🚀 快速启动

### 方式一：双击启动（推荐）

双击 **`启动GitHub工具箱.bat`**，等待浏览器自动打开 `http://localhost:8501` 即可。

### 方式二：创建桌面快捷方式

1. 右键 `启动GitHub工具箱.bat` → **发送到 → 桌面快捷方式**
2. 右键桌面快捷方式 → **属性** → 可修改图标和名称

### 方式三：命令行启动

```bash
cd C:\Users\qwe\Desktop\1234
streamlit_env\Scripts\streamlit run github工具箱_streamlit.py
```

## 📖 功能指南

### 🏠 首页

打开应用后的默认页面，展示：

| 区域 | 说明 |
|------|------|
| **GitHub 账户卡片** | 显示本地 Git 用户名、邮箱、GitHub 用户名 |
| **仓库状态** | ✅ 仓库就绪 / ⚠️ 未配置 / ⚠️ 不可用 |
| **统计面板** | 文件数、文件夹数、提交数、最新 Commit（与 GitHub 一致） |
| **快速入口** | 配置仓库后可一键「打开仓库」 |

### 📤 上传文件到 GitHub

1. 点击左侧导航 **「上传文件」**
2. 点击 **「Choose File」** 选择要上传的文件（可多选）
3. 文件出现在待上传列表中，可单个移除或清空
4. 可选填提交信息（留空自动生成）
5. 点击 **「🚀 开始上传」**
6. 实时显示上传进度：复制 → add → commit → push → 清理

> 💡 上传后默认自动删除本地副本（可在设置中关闭）

### 🗑️ 删除仓库中的文件

1. 点击左侧导航 **「删除文件」**
2. 使用搜索框快速定位文件
3. **勾选**要删除的文件或文件夹（支持批量）
4. 点击 **「☑️ 全选」** 或逐项勾选
5. 点击 **「🗑️ 批量删除」** → 二次确认
6. 实时显示删除进度：rm → commit → push

> ⚠️ 删除操作会同步到远程仓库，不可恢复！

### 🌐 打开仓库

- 在浏览器中打开 GitHub 仓库页面
- 可查看 Git 状态和最近提交记录

### ⚙️ 设置

#### 配置仓库

| 字段 | 说明 | 示例 |
|------|------|------|
| **GitHub 仓库** | 输入 `owner/repo` | `overthinknow/todo_list` |
| **本地仓库路径** | 自动检测或手动填写 | `C:\Users\...\todo_list` |
| **分支名** | 默认 `main` | `main` |

**自动检测路径**：输入 GitHub 仓库名后，系统会自动扫描桌面、文档等常用目录，若找到匹配的 `.git` 仓库则自动填充路径。

**克隆仓库**：若未找到本地仓库，会显示克隆选项：
1. 确认或修改克隆目标路径
2. 系统会在路径末尾自动追加仓库名目录
3. 点击 **「📥 开始克隆」**，等待完成

#### 其他设置

- **上传后自动删除本地副本**：开启后上传成功自动清理本地临时文件

## ❓ 常见问题

**Q: 启动后浏览器没自动打开？**
A: 手动访问 `http://localhost:8501` 即可。

**Q: 提示「未找到仓库」？**
A: 进入「设置」页面，输入 GitHub 仓库名，使用自动检测或克隆功能。

**Q: 上传/删除按钮灰色不可用？**
A: 请先在「设置」中配置并保存仓库信息。

**Q: 如何更换仓库？**
A: 进入「设置」修改 GitHub 仓库和本地路径，保存后即可。

**Q: 端口被占用怎么办？**
A: 修改启动脚本，在命令末尾添加 `--server.port 8502` 使用其他端口。
